#  [![NPM version][npm-image]][npm-url]

> Converts mm / cm / m / inches into pixels


## Install

```sh
$ npm install --save convert-to-pixels
```


## Usage

```js
var pixel-converter = require('convert-to-pixels');

pixel-converter('Rainbow');
```

```sh
# creates a browser.js
$ npm run browser
```
